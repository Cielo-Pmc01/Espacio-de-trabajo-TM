# /copywriter - Generar copy para Meta Ads y RRSS

## Propósito

Generar copy profesional para anuncios de Meta Ads y redes sociales, adaptado a las 14 marcas de Adventure Center y sus diferentes segmentos de audiencia.

## Instrucciones para el asistente

1. **Recopilar brief del usuario**
   - Preguntar: "¿Para qué marca necesitas copy?" (listar las 14 marcas)
   - Preguntar: "¿Qué producto/servicio quieres promocionar?"
   - Preguntar: "¿Cuál es el objetivo?" (venta, leads, awareness, tráfico)
   - Preguntar: "¿Qué plataformas?" (Facebook, Instagram, ambas)
   - Preguntar: "¿Algún tono o estilo específico?"
   - Preguntar: "¿Restricciones de caracteres?"

2. **Cargar contexto de marca**
   - Buscar información de la marca en `contexto/`
   - Extraer:
     - Tono de voz
     - Público objetivo
     - Propuesta de valor única
     - Palabras clave de marca

3. **Generar copy**
   Para cada pieza solicitada:
   - Headline principal (atención + beneficio)
   - Headline alternativas (2-3 opciones)
   - Primary text/cuerpo (con estructura AIDA o PAS)
   - Call-to-action (3 opciones diferentes)
   - Descripción corta (para anuncios)

4. **Adaptar por plataforma**
   - **Facebook:** permite texto más largo, storytelling
   - **Instagram:** visual + caption conciso, hashtags relevantes
   - **Stories/Reels:** corto, impactante, swipe-up CTA

5. **Optimizar para conversión**
   - Usar power words específicas del nicho
   - Incluir elementos de urgencia o escasez si aplica
   - Mentionar beneficios, no solo características
   - Social proof si está disponible

## Estructura de salida

```
## Copy para [Marca] - [Producto/Servicio]

### 📱 Facebook Ads

**Headline:**
[Texto principal]

**Headlines alternativos:**
1. [Alternativa 1]
2. [Alternativa 2]
3. [Alternativa 3]

**Primary Text:**
[Cuerpo del anuncio]

**CTA:**
1. [Opción 1]
2. [Opción 2]
3. [Opción 3]

### 📸 Instagram Ads

**Headline:**
[Texto adaptado]

**Caption:**
[Texto para el feed]

**Hashtags:**
[3-5 hashtags relevantes]

### 📊 Especificaciones técnicas

- **Caracteres headline:** [X/Y]
- **Caracteres primary text:** [X/Y]
- **Recomendación de imagen/video:** [tipo de creativo]
```

## Notas adicionales

- Mantener coherencia con la identidad de marca
- Incluir emojis solo si la marca los usa
- Adaptar lenguaje al público objetivo específico
- Sugerir testing A/B cuando hay múltiples opciones
