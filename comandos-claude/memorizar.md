# memorizar - Guardar información importante

## Descripción

Crea una memoria persistente con información relevante del usuario, preferencias o contexto que deba recordarse en futuras sesiones.

## Instrucciones

1. Preguntar: "¿Qué información quieres que recuerde?"
2. Determinar tipo:
   - Preferencia de usuario
   - Contexto técnico
   - Decisión arquitectónica
   - Información de negocio
3. Verificar duplicados en memoria existente
4. Crear memoria con:
   - Título descriptivo
   - Contenido claro
   - Tags relevantes
   - UserTriggered: true
5. Confirmar al usuario con resumen

## Ejemplos

- "Memoriza que prefiero español"
- "Guarda que usamos React + Node"
- "Recuerda que el tono es joven y aventurero"
