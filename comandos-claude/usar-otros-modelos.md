# usar-otros-modelos - Guía para modelos alternativos

## Descripción

Proporciona estrategia para usar el workspace con modelos de IA con límite de tokens o agentes alternativos.

## Instrucciones

1. Explicar estrategia de tokens:
   - No procesar todo el repo a la vez
   - Trabajar por archivos pequeños
   - Enviar solo fragmentos necesarios

2. Mostrar flujo recomendado:
   - Leer CLAUDE.md primero
   - Revisar contexto/*.md relevante
   - Crear plan en planes/
   - Generar en salidas/

3. Ofrecer crear prompt optimizado:
   - Preguntar si necesita prompt para modelo específico
   - Si sí, usar comando /crear-prompt-modelo-alternativo
