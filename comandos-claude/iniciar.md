# iniciar - Iniciar sesión de trabajo

## Descripción

Lee el contexto del workspace y proporciona un resumen de la estructura, planes activos y contexto disponible para comenzar a trabajar.

## Instrucciones

1. Leer el archivo `CLAUDE.md` si existe en la raíz del proyecto para entender la estructura
2. Leer `README.md` para obtener información general del workspace
3. Listar los directorios principales: `contexto/`, `planes/`, `salidas/`, `referencia/`
4. Identificar archivos `.md` recientes o relevantes en cada directorio
5. Mostrar un resumen estructurado incluyendo:
   - Descripción breve del workspace
   - Lista de planes activos (archivos en `planes/`)
   - Temas principales disponibles en `contexto/`
   - Sugerencias de próximos pasos

## Formato de salida

Presentar el resumen en formato markdown con secciones claras:

```markdown
## Resumen del Workspace

**Estructura:** [breve descripción]

**Planes activos:**
- [lista]

**Contexto disponible:**
- [temas]

**Próximo paso:**
¿Qué te gustaría trabajar hoy?
```
