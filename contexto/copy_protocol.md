# Protocolo de Copywriting y Comunicación Positiva

Este documento establece los criterios obligatorios para la redacción de todo contenido para las marcas de Adventure Center.

---

## Principio Fundamental
Toda comunicación debe transmitir **emociones positivas, confianza, entusiasmo y deseo**.

## Reglas de Oro del Lenguaje
1.  **Eliminar Negatividad**: Evitar palabras que sugieran miedo, duda o peligro (ej. "no es peligroso").
2.  **Enfoque en Soluciones**: En lugar de decir "sin preocupaciones", decir "con todo resuelto para vos".
3.  **Simplicidad y Humanidad**: Usar vocabulario directo y cercano.
4.  **Primera Persona**: Hablar siempre desde la marca (ej. "te acompañamos").

## Palabras y Enfoques PROHIBIDOS vs. RECOMENDADOS

| ❌ EVITAR (Carga Negativa) | ✅ USAR (Refuerzo Positivo) |
| :--- | :--- |
| "No tengas miedo" | "Viví la emoción con total confianza" |
| "No es peligroso" | "Todo está pensado para que lo disfrutes" |
| "No incluye..." | "Para tu comodidad, recordá traer..." |
| "Sin riesgos" | "Te acompañan guías profesionales" |
| "Seguridad garantizada" | "Con el respaldo de expertos" |
| "No te lo pierdas" | "Vivilo hoy / Reservá tu lugar" |
| "Problema / Caída / Golpe" | "Aventura / Desafío / Emoción" |

## Tonos por Sub-Marca de Rafting
- **Rafting Adventure (Madre)**: Enérgico, inclusivo, inspirador.
- **Valle del Manso**: Cálido, natural, contemplativo.
- **Villegas**: Energético, divertido, espontáneo (para jóvenes).
- **Rafting Patagonia**: Sobrio, potente, profesional (para expertos).
- **Rafting Bariloche**: Claro, amable, entusiasta (para primerizos).

## Estándar de Respuesta en WhatsApp/Bot
- Respuestas rápidas, humanas y siempre orientadas a la acción positiva.
- No usar frases corporativas rígidas.
- Cerrar siempre con una invitación o gancho emocional.
