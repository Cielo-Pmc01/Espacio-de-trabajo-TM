# Biblioteca de Prompts Especializados: Excursiones Multi-marca

Este archivo contiene los "Super Prompts" para que Trae actúe como tu brazo derecho en marketing.

## 1. Copywriting para Excursiones (Elementor/Web)
**Objetivo**: Generar descripciones que conviertan visitas en reservas.

```markdown
Actúa como un Copywriter experto en turismo de lujo/aventura. Escribe una descripción de 300 palabras para la excursión [NOMBRE_EXCURSION] de la marca [MARCA]. 
- Incluye: Gancho inicial, itinerario resumido, qué incluye, y una sección de '¿Por qué elegirnos?'.
- Tono: [TONO_MARCA]
- Keywords SEO: [KEYWORDS]
- CTA: Enlace directo al booking.
```

## 2. Creative Brief para Diseñador (Logo/Video)
**Objetivo**: Dar instrucciones claras y técnicas al equipo creativo.

```markdown
Genera un briefing detallado para nuestro Diseñador Gráfico sobre una campaña de Meta Ads para [MARCA]. 
- Tipo de Contenido: [Video 15s / Carrusel / Logo]
- Objetivo: [Conversión / Awareness]
- Elementos Clave: [Ej. Logo de marca, colores corporativos, tipografía, tomas de drones].
- Gancho visual sugerido: [Ej. Primeros 3 segundos con impacto].
```

## 3. Planificación para Community Manager (Calendario)
**Objetivo**: Automatizar la creación de contenido semanal.

```markdown
Crea un calendario de contenido de 7 días para las redes sociales de [MARCA]. 
- Lunes: Tip de viaje.
- Miércoles: User Generated Content (UGC) sugerido.
- Viernes: Promo de fin de semana.
- Domingo: Curiosidad del destino.
Incluye: Copy del post, 10 hashtags estratégicos y descripción del reel sugerido.
```

## 4. Precios Dinámicos y Temporada
**Objetivo**: Ajustar ofertas según la demanda.

```markdown
Analiza la temporada de [DESTINO] para el mes de [MES]. 
- Sugiere una estrategia de 'Early Bird' o 'Last Minute' para la marca [MARCA].
- Genera el copy promocional para el banner de la web en Elementor.
```
