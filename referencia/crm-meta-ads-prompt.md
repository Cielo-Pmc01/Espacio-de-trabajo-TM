# Prompt para crear / mejorar el CRM TM

Actuá como experto en CRM y gestión comercial para empresas de turismo.

Necesito que me ayudes a diseñar un CRM para un área de marketing (TM) que genera leads desde Meta Ads para excursiones en Bariloche.

El CRM tiene que:

- Permitir seguimiento de leads desde que ingresan hasta que compran o se pierden
- Mostrar claramente el rendimiento de campañas (leads, ventas, conversión)
- Ser simple de usar para equipo operativo (ventas/SP)
- Dar visibilidad clara para dirección (reportes semanales)

Contexto:
- Tenemos múltiples marcas (TC, TB, etc.)
- Vendemos excursiones (Piedras Blancas, Circuito Chico, etc.)
- Los leads entran principalmente por WhatsApp
- El problema actual es falta de seguimiento y falta de datos reales de conversión

Necesito que me armes:

1. Estructura del pipeline ideal
2. Campos obligatorios que debe tener cada lead
3. Cómo organizar la información en una tabla (tipo Google Sheets o CRM)
4. Qué métricas debo medir sí o sí
5. Cómo debería ser un reporte semanal para dirección
6. Errores comunes a evitar en implementación

Sé concreto, estructurado y orientado a resultados.
